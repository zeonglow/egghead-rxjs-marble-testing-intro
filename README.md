# Egghead.io Intro to RxJS marble testing Redux

Code to accompany [egghead.io](https://egghead.io) Intro to RxJS marble testing lesson.

Tweaked to run with newer version of RxJS library

### Getting Started

1. (Optional, but recommended)

Install [Volta](https://volta.sh/) follow their instructions

2. Run the tests

```bash

# clone the repo
git clone https://github.com/zeonglow/egghead-rxjs-marble-testing-intro.git

# run install
yarn install
yarn test
```
